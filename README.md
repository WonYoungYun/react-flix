React로 만드는 무비 웹
=


리액트를 사용한 영화 API를 이용한 무비 웹


페이지
-
- 영화
- 쇼 프로그램
- 검색
- 자세한 소개

API
[themoviedb](https://www.themoviedb.org/)의 API를 이용하여 제작하였습니다.

설치방법
-

```
yarn install
```

src/config/index.js를 생성해야 합니다.
```
const cfg = {
    key: ""//themoviedb의 api KEY
}

export default cfg
```

실행
-
```
yarn start
```
